nesDS
-=-=-=-=-=-=-=-

It's a NES emulator for the Nintendo DS.  Hooray!

nesDS must be patched with a DLDI driver to work properly.
For more information, go to http://dldi.drunkencoders.com/

Usage
-=-=-=-=-=-=-

Touch the screen to bring up the ingame menu.

L,R buttons are rewind / fast forward controls.

X,A buttons are autofire for Y,B.
